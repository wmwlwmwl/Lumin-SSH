操作步骤

  1. 在 https://www.kimi.com/ 完成网站登录。
  2. 打开 F12 开发者控制台。
  3. 在控制台手动输入以下内容并分别回车：
    允许粘贴
    allow pasting
  4. 执行下面这段 JavaScript，将当前页面的全部本地存储复制到剪贴板：
    (() => { const data = {}; for (let i = 0; i < localStorage.length; i += 1) { const key = localStorage.key(i); data[key] = localStorage.getItem(key); } const text = JSON.stringify(data, null, 2); const textarea = document.createElement('textarea'); textarea.value = text; textarea.setAttribute('readonly', ''); textarea.style.position = 'fixed'; textarea.style.top = '-1000px'; textarea.style.left = '-1000px'; document.body.appendChild(textarea); textarea.focus(); textarea.select(); textarea.setSelectionRange(0, textarea.value.length); const copied = document.execCommand('copy'); document.body.removeChild(textarea); console.log(copied ? '已复制 localStorage 到剪贴板' : '复制失败，请手动全选复制下面内容：'); if (!copied) { console.log(text); } })();
  5. 粘贴到Apikey输入框
